## 🎓 Comment ça Marche?

**Recherche:**
```
User → Adresse → Nominatim geocode → Lat/Lon
                   ↓
                Search API → KV cache → Stations
                   ↓
          Score + Rank → Top 10
                   ↓
              Afficher résultats
```

**Scoring:**
```
Pour chaque station:
  - Normaliser prix (moins = mieux)
  - Normaliser distance (moins = mieux)
  - Score = (price * 70% + distance * 30%) * 100
  - Trier descending
```

**Cache:**
```
Browser (60s) → KV (3600s) → XLSX refresh
```

Simple, efficace, scalable! ⚡
